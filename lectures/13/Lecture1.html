مجموع دو متغیر تصادفی دوجمله‌ای:

فرض کنید $X$ و $Y$ دو متغییر تصادفی مستقل با توزیع دوجمله‌ای باشند:

$$ X \sim Bin(n_1,p) \:\:\:,\:\:\: Y \sim Bin(n_2,p) $$

حاصل جمع این دو متغیر هم دارای توزیع دوجمله‌ای خواهد بود:

$$X + Y \sim Bin(n_1+n_2,p)$$

شهود: $X$ از $ n_1 $ آزمایش و $Y$ از $ n_2 $ آزمایش تشکیل شده است.

هر آزمایش دارای احتمال موفقیت $p$ است.

متغیر تصادفی $Z$ را برابر تعداد موفقیت‌ها در $ n_1+n_2 $ آزمایش درنظر می‌گیریم.

واضح است که $Z$ دارای توزیع دوجمله‌ای $Bin(n_1+n_2,p)$ است و از سوی دیگر:

$$ Z = X + Y $$

مجموع دو متغیر تصادفی دوجمله‌ای:

در حالت کلی اگر $X_i$ها متغیر‌های تصادفی دوجمله‌‌ای مستقل با توزیع $Bin(n_i,p)$ باشند، $(1 \leq i \leq N)$ آنگاه با استدلال مشابهی داریم:

$$ \left(\sum_{i=1}^{N} X_i \right) \sim Bin \left(\sum_{i=1}^{N} n_i,p \right)$$

حالت خاص: اگر $X_i$ها متغیر تصادفی برنولی با احتمال موفقیت $p$ باشند: $X_i \sim Ber(p)$

توزیع برنولی حالت خاص توزیع دوجمله‌ای برای $ n = 1 $ است.

حاصل جمع $X_i$ها دارای توزیع دوجمله‌ای $ Bin(N,p) $ خواهد بود.

مجموع دو متغیر تصادفی پواسون:

فرض کنید $X$ و $Y$ دو متغیر تصادفی مستقل با توزیع پواسون باشند:

$$X \sim Poi(\lambda_1) \: , \: Y \sim Poi(\lambda_2)$$

حاصل جمع $X$ و $Y$ نیز دارای توزیع پواسون خواهد بود:

$$ X + Y \sim Poi(\lambda_1 + \lambda_2) $$

مجموع دو متغیر تصادفی مستقل پیوسته:

فرض کنید $X$ و $Y$ دو متغیر تصادفی مستقل باشند. تابع توزیع انباشته (CDF) متغیر تصادفی $Y+Y$ برابر است با:

$$F_{X+Y}(a) = P(X+Y \leq a) = \int_{-\infty}^{\infty} F_X(a-y)f_Y(y)dy$$

با مشتق گیری نسبت به $a$ داریم:

$$f_{X+Y}(a) = \int_{-\infty}^{\infty} f_X(a-y)f_Y(y)dy$$

انتگرال فوق را کانولوشن (convolution) یا همگشت دو تابع $f_X$ و $f_Y$ می‌نامیم.

مجموع دو متغیر تصادفی مستقل گسسته:

در حالت گسسته انتگرال کانولوشن تبدیل به جمع کانولوشن می‌شود:

$$ P_{X+Y} = \sum_{y=-\infty}^{+\infty} P_X(a-y)P_Y(y)$$

مجموع دو متغیر تصادفی مستقل یکنواخت:h3>

$X$ و $Y$ دو متغیر تصادفی مستقل با توزیع U(0,1) هستند:

$$ f_X(x) = 1 : 0 < x < 1 \:\: , f_Y(y) = 1 : 0 < y < 1 $$

$$f_{X+Y}(a) = \int_{-\infty}^{\infty} f_X(a-y)f_Y(y)dy = \int_{0}^{1}f_X(a-y)dy$$$$ a -1 < y < a $$

از طرفی $ 0 < y < 1 $ و از طرف دیگر $ a-1 < y < a $، بنابراین برای داشتن محدوده صحیح $Y$، باید محدوده $a$ را به دو قسمت تقسیم کنیم:

حالت اول:

$$ 0 < a < 1 \rightarrow 0 < y < a $$$$f_{X+Y}(a) = \int_{0}^{a} f_X(a - y) dy = a$$

حالت دوم:

$$ 1 < a < 2 \rightarrow a-1 < y < 1 $$$$f_{X+Y}(a) = \int_{a-1}^{1} f_X(a - y) dy = 2 - a$$

درنتیجه:

$$f_{X+Y}(a) = \begin{cases} a \:\:\:\:\:\:\:\:\:\:\:\:\:\:\:\: 0 < a < 1 \\ 2 - a \:\:\:\:\:\:\:\:\: 1 < a < 2 \end{cases}$$

جمع دو متغیر تصادفی مستقل نمایی:

$X$ و $Y$ دو متغیر تصادفی مستقل با نوزیع $Exp(\lambda)$ هستند:

$$ f_X(x) = \lambda e ^ {-\lambda x}\: : \: x > 0 \:\:\: , f_Y(y) = \lambda e ^ {-\lambda y} \: : \: y > 0$$

برای متغیر تصادفی $ Z = X + Y $ داریم:

$$ f_Z(z) = \int_{-\infty}^{\infty} f_X(z-y)f_Y(y)dy = \lambda^2 z e ^{-\lambda z} \:\:\:\:\: 0 < y < z$$

واریانس مجموع متغیرهای تصادفی مستقل:

دیدیم که برای متغیرهای تصادفی $X_i$ (چه مستقل چه پیوسته) داریم:

$$E[X_1 + X_2 + ... + X_n] = E[X_1] + E[X_2] + ... + E[X_n]$$

برای متفیرهای تصادفی $X_i$ در صورتی که مستقل از هم باشند، داریم:

$$Var(X_1 + X_2 + ... + X_n) = Var(X_1) + Var (X_2) + ... + Var(X_n)$$

در حالت کلی اگر $X_i$ ها مستقل باشند:

$$Var(a_1X_1 + a_2X_2 + ... + a_nX_n) = a_1^2 Var(X_1) + a_2^2 Var (X_2) + ... + a_n^2 Var(X_n)$$

و در همه حالات (چه $X_i$ها مستقل باشند و چه وابسته):

$$E[a_1X_1 + a_2X_2 + ... + a_nX_n] = a_1E[X_1] + a_2E[X_2] + ... + a_nE[X_n]$$

مجموع دومتغیر تصادفی مستقل نرمال:

فرض کنید $X$ و $Y$ متغیرهای تصادفی مستقل نرمال باشند:

$$ X \sim N(\mu_1 , \sigma_1^2) \:\:\:\:\: , \:\:\:\:\: Y \sim N(\mu_2 , \sigma_2^2)$$

در این صورت حاصل جمع X + Y نیز دارای توزیع نرمال خواهد بود:

$$X + Y \sim N(\mu_1 + \mu_2 , \sigma_1^2 + \sigma_2^2)$$

در حالت کلی، اگر n متغیر تصادفی مستقل نرمال $X_i \sim N(\mu_i , \sigma_i^2)$ داشته باشیم:

$$ \left(\sum_{i=1}^{n} X_i \right) \sim N \left(\sum_{i=1}^{n} \mu_i \:,\: \sum_{i=1}^{n} \sigma_i^2 \right)$$
