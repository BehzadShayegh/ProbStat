متغییر تصادفی گسسته

متغییر تصادفی X را گسسته(discrete) گویند اگر $F_X(x)$ (تابع توزیع انباشته ی آن) به صورت پلکانی باشد.
در این حالت داریم:

$ p_i = P\{X = x_i\} = F(x_i) - F(x_i^{-}\!)$

این تعریف معادل با این است که مقادیر ممکنه برای X قابل شمارش باشند.

متغییر تصادفی پیوسته

متغییر تصادفی X را پیوسته(continuous) گویند اگر $F_X(x)$ برای هر x پیوسته باشد.
با توجه به ویژگی های تابع توزیع انباشتف، برای متغییر تصادفی پیوسته X، نتیجه میگیریم که برای هر عدد a داریم:

$ P\{X = a\} = F(a) - F(a^{-}\!) = 0$

برای محاسبه احتمال پیشامدهای مربوط به یک متغییر تصادفی پیوسته می توانیم از تابع توزیع انباشته آن استفاده کنیم:

$P\{a \leq X \leq b\} = F_X(b) - F_X(a)$

متغییر تصادفی مخلوط

متغییر تصادفی X را از نوع مخلوط(mixed) می گویند، اگر $F_X(x)$ دارای ناپیوستگی باشد
اما به صورت پلکانی نباشد(ترکیبی از حالات گسسته و پیوسته).
برخی مراجع متغیر تصادفی پیوسته را متغیری تعریف می کنند که مقادیر ممکن برای
آن غیرقابل شمارش است. این تعریف هر دو نوع پیوسته و مخلوط را با هم دربر میگیرد.

تابع چگالی احتمال (Probability Density Function)

برای متغیر تصادفی پیوسته راه دیگر برای مشخص کردن احتمال پیشامدهایی که
متغیر تصادفی را تعریف می کنند، آن است که تابعی که بیانگر چگالی احتمال است را داشته باشیم.
به وسیله pdf دید بیشتری )در مقایسه با CDF) نسبت به میزان محتمل بودن مقادیر مختلف پیدا میکنیم.
pdf را به صورت مقابل تعریف میکنیم:

$f_X(x) = \frac{dF_X(x)}{dx}$

نکته: معمولا CDF را تابع توزیع و pdf را نوزیع متغییر تصادفی X می گویند.

خواص تابع چگالی احتمال

  1. $ P\{x1 \leq X \leq x2\} = \int_{x1}^{x2} f_X(x) \,dx $
    میتوان نتیجه کرفت:

    $\normalsize f_X(x) = \lim_{\Delta x \to 0} \large\frac{P\{x \leq X \leq x+\Delta x\}}{\Delta x} $

    نکته: $f_X(x)$ بر خلاف $P_X(x)$ یک احتمال نیست و احتمال در یک واحد X (چگالی) است، بنابر این
    میتواند مقادیر بزرگتر از یک اختیار کند.

    همچنین داریم:
    $F_X(a) = \int_{-\infty}^a f_X(x)dx $

  2. $\forall x : f(x) \geq 0 $

  3. $\int_{-\infty}^\infty f(x)dx = 1 $
