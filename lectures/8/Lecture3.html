توزیع یکنواخت پیوسته

گوییم متغییر تصادفی X دارای توزیع یکنواخت بین a و b است، $X~U(a,b)$، اگر:

$f(x) = \large \frac{1}{b-a} \small \ \ a \leq x \leq b \ ,\ otherwise :\ 0$

داریم:
$P\{\alpha < X < \beta\} =\large \frac{\beta \ - \ \alpha}{b \ - \ a} \small \ \ \ a \leq \ \alpha , \beta \ \leq b $

$Var(x) =\large \frac {(b - a)^2}{12}$

قضیه:
$E(X) = \int_{0}^\infty P(X \geq x)dx = \int_{0}^\infty (1-F_X(X))dx $
